# 머신러닝 : Machine Learning

## Python을 이용한 머신러닝

## 경사하강법

### 회귀기법 
  - 선형회귀, 로지스틱 회귀

### 확률기반 
  - 나이브 베이즈 분류기, 은닉 마르코프 모델

### 기하기반 
  - k-Means Clustering(kMC), k-Nearest Neighbors(kNN)
    - Support Vector Machine(SVM)

### 인공신경망 
  - Perceptron, Multi Layer Perceptron, Deep Neural Network
